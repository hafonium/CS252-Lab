## Running backend server:

### On local machine:
Install modules:
```bash
cd backend
uv sync
```
Run:
```bash
uv run main.py
```

## Running frontend server:
Install modules:
```bash
npm install 
```
After installation completing, the server is runnable:
```bash
npm run dev
```
Now, the website is running at http://localhost:5173/
